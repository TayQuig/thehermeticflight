# The Hermetic Flight
Astro-based tarot and esoteric content website with blog, quiz, and FAQ pages.

## Folder Structure
```
src/            → Astro components, pages, content collections, layouts, styles
public/         → Static assets (images, favicons)
dist/           → Vercel build output
docs/           → Documentation
scripts/        → Utility scripts (webhook testing)
operations/     → TASKBOARD.md, task-archive.md
reference/      → Source material, research, inputs
deliverables/   → Outputs and finished work
archive/        → Superseded docs, historical reference
```

## Tech Stack
- **Framework:** Astro 5.16.4 (SSG) with Vercel adapter
- **Styling:** Tailwind CSS 3.4 + Typography plugin
- **Content:** Astro Content Layer with MDX support
- **Deploy:** Vercel
